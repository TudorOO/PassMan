PASSMAN

Flask-based password manager


Passman password manager provides a safe way to create and store incredibly strong passwords for all of your apps. It provides safety and convinience, removing the need to remember an ever-growing number of passwords, while also making sure that they are complex enough to be safe in the event of a breach.



Why it's safe:


1. The master password is stored on the server for virtually no time(or actually no time).

This makes sure that, even if our server is breached, your passwords will still be safe.


2. The passwords are stored using the latest encryotion technology available.

We hash the master password with Argon2id and encrypt the passwords using a key derived from the master password with AESGCM. These technologies provide powerful protection against even the strongest attacks, such as GPU-Based ones.


3. Encryption and Decryption of passwords can happen either only client-side, or only server-side.

You get to choose! Do you trust your machine more or our server more? Whichever the case, you can have it your way: zero-knowledge or minimal-knowledge!


4. No AI corners cut.

Every line of code in this app has been created by a human. We believe that AI is not yet fit to handle such vulnerable data as passwords, and as such we have decided not to use it in our development process.



How it works:


1. You make an account

The master password you provide is encrypted using Argon2id and stored. In addition, a backup file is generated, which you can then download and use if you wish to login without a password later.

WARNING: While the backup file is encrypted, we do not provide any protection in the case that a nefarious actor gets a hold of it. You should store it incredibly safely, ideally in an USB or another location not connected to the internet.
